KlarnaHybridSDK

public final class KlarnaHybridSDK : NSObject

Klarna’s solution to integrating it’s products in a hybrid context (one in which Klarna’s) products are presented inside a merchant-owned web view.

  • shouldFollowNavigation(withRequest:)

    Verify with the SDK whether a web view navigation should be followed in your web view.

    If using a UIWebView, checking should be performed in: webView(_: shouldStartLoadWith: navigationType:)

    If using a WKWebView, checking should be perfomed in: webView(_: decidePolicyFor: decisionHandler:) wuth the navigationAction‘s reqeuest propeerty.

    Note

    If it is a page Klarna recognizes as one of its own (e.g. Klarna’s financing terms), the SDK will return false. You should block this navigation.

    If it’s a page Klarna doesn’t recognize, it’ll fall back to returning true. Your app should determine whether it wants to load the URL through its own heuristics.
